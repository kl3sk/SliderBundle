SliderBundle
============

Create slider using coin-slider and Gregwar-ImageBundle

### HowTo : Comming soon


## config
    klesk_slider:
      width: 930
      height: 255
      square_per_width: 7
      square_per_height: 5
      delay: 3000
      effect: random
      next: Next
      prev: prev
      resize: none
      imagepath: "images/news/"

> if `resize`if different than `none`. It use GregwarImageBundle
